슈팅게임 만들기

비행기를 모두 격추하면 승리, 그렇지 못하면 패배
목숨 3개
내 캐릭터는 상하좌우 전부 이동이 가능하며, 무기를 발사.
무기의 종류는 총알(30발)-space, 
아이템은 상대 비행기를 격추하면 일정 확률(50%)로 등장
아이템을 먹으면 총알 10발/ 목숨(1개) 획득


상대 비행기는 큰 비행기부터 작은 사이즈까지 3개의 종류
격추되면 일정 확률로 아이템 드랍
큰 비행기가 격추되면 한단계 작은 사이즈의 비행기 2대가 등장.
각 비행기는 일직선으로 날라가는 총알만 발사, 이것에 맞으면 게임 종료
비행기에 충돌해도 게임 종료

게임 종료 조건
성공
모두 격추 완료 - Mission Complete
실패
시간초과 - time out
격추 당함 - Mission Failed



총알 없음 - No Bullet
